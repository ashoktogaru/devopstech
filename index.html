PHASE 01
Linux & Command Line
2–3 weeks
The foundation of DevOps. Everything runs on Linux — learn it deeply.
✓ File System Navigation
✓ Users & Permissions
✓ Package Management
✓ Processes & Jobs
✓ Bash Scripting
✓ Networking Basics
✓ File Editors (vim/nano)
✓ Cron Jobs
🔬 HANDS-ON LABS
Set up Ubuntu VM using VirtualBox or WSL2
Write a bash script to automate system backup
Configure user permissions and sudoers
Build a log monitoring script with grep & awk
🌐
PHASE 02
Networking & Security Basics
2 weeks
Understand how data moves across networks and how to secure it.
✓ TCP/IP, DNS, HTTP
✓ Firewalls & iptables
✓ SSH & Key Pairs
✓ Load Balancing
✓ SSL/TLS Certificates
✓ VPN Concepts
🔬 HANDS-ON LABS
Configure SSH key-based authentication
Set up UFW firewall rules on Ubuntu
Use curl & netcat to inspect HTTP traffic
📦
PHASE 03
Git & Version Control
1–2 weeks
Master Git — the core of collaborative software development.
✓ Git Fundamentals
✓ Branching Strategies
✓ Pull Requests
✓ Merge & Rebase
✓ GitHub / GitLab
✓ Git Hooks
🔬 HANDS-ON LABS
Create a GitHub repo and practice branching
Simulate a team workflow with PRs and code review
Write a pre-commit hook for code linting
🐳
PHASE 04
Docker & Containers
3 weeks
Containerization changed everything. Learn Docker inside-out.
✓ Docker Architecture
✓ Dockerfile Best Practices
✓ Docker Compose
✓ Volumes & Networks
✓ Container Registries
✓ Multi-stage Builds
✓ Docker Security
🔬 HANDS-ON LABS
Containerize a Node.js/Python web application
Build multi-service app with Docker Compose
Push image to Docker Hub / GitHub Container Registry
Optimize Dockerfile for smaller image size
⚙️
PHASE 05
CI/CD Pipelines
3 weeks
Automate your build, test, and deployment pipeline — the heart of DevOps.
✓ CI/CD Concepts
✓ GitHub Actions
✓ Jenkins
✓ GitLab CI
✓ Pipeline Stages
✓ Secrets Management
✓ Artifact Management
🔬 HANDS-ON LABS
Build GitHub Actions workflow: test → build → push
Set up Jenkins pipeline with Jenkinsfile
Implement blue-green deployment strategy
☁️
PHASE 06
Cloud Platforms (AWS / GCP / Azure)
4 weeks
Master cloud infrastructure — start with AWS, the industry standard.
✓ EC2 & VPC
✓ S3 & Storage
✓ IAM & Security
✓ RDS & Databases
✓ Lambda & Serverless
✓ EKS / GKE
✓ CloudWatch & Logs
🔬 HANDS-ON LABS
Deploy a web app on EC2 behind ALB
Configure IAM roles and policies
Set up S3 static hosting with CloudFront CDN
Build a Lambda function triggered by S3 events
🏗️
PHASE 07
Infrastructure as Code
3 weeks
Manage infrastructure with code using Terraform and Ansible.
✓ Terraform Basics
✓ Terraform State
✓ Modules & Workspaces
✓ Ansible Playbooks
✓ Ansible Roles
✓ Pulumi Concepts
🔬 HANDS-ON LABS
Provision AWS VPC + EC2 with Terraform
Use Ansible to configure Nginx on remote servers
Store Terraform state in S3 + DynamoDB locking
☸️
PHASE 08
Kubernetes & Orchestration
4 weeks
The gold standard for container orchestration at scale.
✓ K8s Architecture
✓ Pods & Deployments
✓ Services & Ingress
✓ ConfigMaps & Secrets
✓ Helm Charts
✓ Auto-scaling
✓ Persistent Volumes
✓ RBAC & Security
🔬 HANDS-ON LABS
Deploy app on Minikube / kind locally
Create Helm chart for a microservice
Implement HPA (Horizontal Pod Autoscaler)
Set up Ingress with SSL using cert-manager
📊
PHASE 09
Monitoring & Observability
2 weeks
You can't manage what you can't measure. Build full observability stacks.
✓ Prometheus
✓ Grafana Dashboards
✓ ELK Stack
✓ Jaeger / Tracing
✓ Alerting & On-call
✓ SLI / SLO / SLA
🔬 HANDS-ON LABS
Deploy Prometheus + Grafana with Docker Compose
Build a K8s cluster dashboard in Grafana
Set up alerting rules and PagerDuty integration
🛡️
PHASE 10
DevSecOps & Advanced Topics
3 weeks
Security-first DevOps, GitOps, and the skills that make you stand out.
✓ DevSecOps Principles
✓ SAST / DAST Tools
✓ GitOps with ArgoCD
✓ Service Mesh (Istio)
✓ Chaos Engineering
✓ Cost Optimization
🔬 HANDS-ON LABS
Set up ArgoCD for GitOps-based deployment
Run Trivy container vulnerability scanner in CI
Implement chaos experiments with Chaos Monkey
